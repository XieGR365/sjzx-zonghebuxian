# 用户故事 US-003：查看布线记录列表

## 1. 用户角色
系统用户

## 2. 用户故事描述
作为一个系统用户，我希望能够查看系统中所有的布线记录列表。

## 3. 验收标准
### 3.1 功能要求
- [ ] 表格形式展示所有记录
- [ ] 按创建时间倒序排列
- [ ] 支持分页显示（每页20条）
- [ ] 点击记录查看详细信息

### 3.2 界面要求
- [ ] 表格显示：登记表编号、机房名称、用户单位、创建时间
- [ ] 分页控件
- [ ] 刷新按钮

## 4. 技术实现要点
- 使用Element Plus表格组件
- 后端分页查询接口

## 5. 备注
- 快速开发版本，只显示基本信息
