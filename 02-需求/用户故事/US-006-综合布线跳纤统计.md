# 用户故事 US-006：综合布线跳纤统计

## 1. 用户角色
系统用户

## 2. 用户故事描述
作为一个系统用户，我希望能够查看各机房的综合布线跳纤统计信息，包括跳纤数量和线路状态，以便快速了解各机房的布线情况。

## 3. 验收标准
### 3.1 功能要求
- [ ] 按机房分组展示跳纤统计信息
- [ ] 显示每个机房的跳纤总数量
- [ ] 显示每个机房的在用线路数量
- [ ] 显示每个机房的已拆除线路数量
- [ ] 支持按机房名称筛选
- [ ] 支持按线路状态筛选（全部、在用、已拆除）
- [ ] 点击机房模块查看该机房的详细跳纤记录
- [ ] 支持导出统计报表

### 3.2 界面要求
- [ ] 模块化展示各机房统计卡片
- [ ] 每个机房卡片显示：
  - [ ] 机房名称
  - [ ] 跳纤总数量（大字号显示）
  - [ ] 在用线路数量（绿色标识）
  - [ ] 已拆除线路数量（红色标识）
  - [ ] 状态占比进度条（绿色/红色）
- [ ] 卡片网格布局（响应式设计）
- [ ] 顶部筛选区域（机房名称、线路状态）
- [ ] 导出按钮

### 3.3 数据要求
- [ ] 统计数据实时计算
- [ ] 支持缓存优化（可选）
- [ ] 数据更新后自动刷新统计

## 4. 技术实现要点
- 使用数据库聚合查询统计各机房跳纤数量
- 按线路状态（在用/已拆除）分组统计
- 使用Vue 3组合式API实现模块化卡片组件
- 使用Element Plus卡片组件和进度条组件
- 响应式布局（使用CSS Grid或Flexbox）
- 状态颜色标识：在用（绿色#67C23A）、已拆除（红色#F56C6C）

## 5. 备注
- 快速开发版本，只支持基本统计和筛选
- 线路状态字段需要从现有布线记录中获取
- 默认显示所有机房统计
